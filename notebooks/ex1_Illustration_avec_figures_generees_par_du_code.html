2) Courbe paramétrée :

Rappel : Coordonnées cartésiennes d'un point $M$ en fonction de l’angle polaire $\theta$. On obtient :

$$ x = r \cos (\theta) = f(\theta) \cos(\theta) $$$$ y=r \sin (\theta) = f(\theta) \sin(\theta)$$

Exemple 1 : Prenons l'exemple d'une cardioïde, qui est une courbe paramétrée ayant pour paramètre $\theta$ d'équation :

$ \ \ \ \ f : \mathbb{R} \rightarrow \mathbb{R}$

$ \ \ \ \ f : \theta \mapsto r = f(\theta) = a (1+ \cos(\theta))$

On prendra $a=1$ dans cet exemple.

In [3]:
Theta = np.linspace(0, 2*np.pi, 100) # tableau numpy 1D des angles theta
a = 1
R = a*( 1 + np.cos(Theta) ) # tableau numpy 1D des rayons r
X = R*np.cos(Theta) # tableau numpy 1D des abscisses x
Y = R*np.sin(Theta) # tableau numpy 1D des ordonnées y
plt.figure(1)
plt.axis("equal") # axes de longueurs égales
plt.plot(X, Y) # tracé de la courbe polaire
plt.grid(True)
plt.show()

# en python, on trace des champs de vecteurs avec des flèches grace à la fonction "quiver" de matplotlib
%matplotlib inline
X, Y = np.meshgrid(np.arange(0, 2 * np.pi, .2), np.arange(0, 2 * np.pi, .2))
U = np.cos(X)
V = np.sin(Y)

# Champ de vecteur où les fleches sont mises a l'echelle en fonction du graphe
plt.figure()
plt.title('Champ de vecteur')
Q = plt.quiver(X, Y, U, V, units='width')
qk = plt.quiverkey(Q, 0.9, 0.9, 2, r'$2 \frac{m}{s}$', labelpos='E',
                   coordinates='figure')

# Même champ de vecteur mais où on ne trace qu'une flèche sur 3
plt.figure()
plt.title("pivot='mid'; Toutes les 3 fleches ; units='inches'")
Q = plt.quiver(X[::3, ::3], Y[::3, ::3], U[::3, ::3], V[::3, ::3],
               pivot='mid', units='inches')
qk = plt.quiverkey(Q, 0.9, 0.9, 1, r'$1 \frac{m}{s}$', labelpos='E',
                   coordinates='figure')
plt.scatter(X[::3, ::3], Y[::3, ::3], color='r', s=5)
 
plt.show()

$ \ \ \ \ f : (x,y,z) \mapsto (\sin(\pi x) \cos(\pi y) \cos(\pi z), -\cos(\pi x) \sin(\pi y) \cos(\pi z), \sqrt{\frac{2}{3}} \cos(\pi x) \cos(\pi y) \sin(\pi z))$

from mpl_toolkits.mplot3d import axes3d
import matplotlib.pyplot as plt
import numpy as np

fig = plt.figure()
ax = fig.gca(projection='3d')

x, y, z = np.meshgrid(np.arange(-0.8, 1, 0.2),
                      np.arange(-0.8, 1, 0.2),
                      np.arange(-0.8, 1, 0.8))

u = np.sin(np.pi * x) * np.cos(np.pi * y) * np.cos(np.pi * z)
v = -np.cos(np.pi * x) * np.sin(np.pi * y) * np.cos(np.pi * z)
w = (np.sqrt(2.0 / 3.0) * np.cos(np.pi * x) * np.cos(np.pi * y) *
     np.sin(np.pi * z))

# on peut changer la longueur des flèches (quiver en anglais) en changeant la valeur du paramètre length
ax.quiver(x, y, z, u, v, w, length=0.2)

plt.show()
